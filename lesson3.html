Getting our code to do more

How would we write code to print all numbers between 10 and 0?

What about 100? Or 1000?

By using loops, we can make python evaluate code more than once

              number = 10
              
              while number > 0: 
                  print(number)
                  number = number - 1
              
              print("Blast Off!")

While loops

While loops are like if statements,
except they keep looping through their block of code until the Boolean is False

What will this code do? 

              number = 10
              
              while number > 0: 
                  print(number)
                  #  number = number - 1
              
              print("Blast Off!")

Accepting user input


What if we could provide data while our code is running?

input pauses the program until the user types something and presses Enter.

We can then use what the user typed as a variable in the code! 


          users_name = input('What is your name? ')
          print('Hello, ' + users_name + '!')
